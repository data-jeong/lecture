# 02. Databases - 데이터베이스

## 📚 과정 소개

## 🎯 학습 목표

- 광고 데이터 특성에 맞는 DB 선택
- 대용량 데이터 처리 및 실시간 분석
- 데이터베이스 성능 튜닝
