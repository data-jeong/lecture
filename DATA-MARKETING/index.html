왜 텍소노미가 중요한가?

"측정할 수 없으면 개선할 수 없다" - 피터 드러커

어떤 블로그 카테고리가 가장 많은 뉴스레터 구독을 유도하는가? 블로그 게시물 하단의 '구독' 버튼 클릭 generate_lead 클릭 트리거, CSS 선택자 .blog-subscribe-btn form_location: 'blog_post_footer'
content_category: {{DLV - postCategory}}		 Form Location
Content Category
어떤 프로모션 배너가 가장 많은 클릭을 유도하는가? 메인 페이지의 프로모션 배너 클릭 select_promotion 클릭 트리거, Click Element는 CSS 선택자 .promo-banner와 일치 promotion_name: {{DLV - promoName}}
creative_slot: 'main_banner_1'		 Promotion Name
Creative Slot
사용자들이 회원가입 과정에서 어느 단계에서 가장 많이 이탈하는가? 회원가입 양식의 각 필드 입력 완료 후 다음 버튼 클릭 signup_progress 클릭 트리거, 각 단계별 버튼 ID signup_step: 'step_2_verify_email' Signup Step
