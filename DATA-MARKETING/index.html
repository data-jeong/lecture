어떤 블로그 카테고리가 가장 많은 뉴스레터 구독을 유도하는가? 블로그 게시물 하단의 '구독' 버튼 클릭 generate_lead 클릭 트리거, CSS 선택자 .blog-subscribe-btn form_location: 'blog_post_footer'
content_category: {{DLV - postCategory}}		 Form Location
Content Category

creative_slot: 'main_banner_1'		 Promotion Name
Creative Slot

📋 문제 정의

해결하고자 하는 비즈니스 문제는 무엇인가?

예: 모바일 사용자의 구매 전환율이 데스크톱 사용자보다 30% 낮다.
💡 가설

"우리는...라고 믿는다" 형식의 완전한 문장

예: 우리는 모바일 결제 프로세스를 3단계에서 1단계로 줄이면, 모바일 구매 전환율이 15% 증가할 것이라고 믿는다.
👥 타겟 잠재고객/세그먼트

이 테스트는 누구를 대상으로 하는가?

예: 모든 모바일 기기 사용자
📊 성공 지표 (주요 & 가드레일)

변화를 기대하는 주요 KPI는 무엇인가? 악화되어서는 안 되는 KPI는 무엇인가?

예: 주요 지표: 세션 전환율, 가드레일 지표: 평균 주문 가치
🔧 실행 세부사항

정확히 무엇을 변경/구축해야 하는가?

예: 원클릭 결제 버튼 도입, GTM을 통해 결제 완료 이벤트 추적
📈 결과

테스트에서 얻은 정량적 데이터

예: 실험군 전환율 3.5%, 대조군 전환율 3.1%, 통계적 유의성 95%
✅ 결론 및 다음 단계

가설이 검증되었는가? 무엇을 배웠는가? 전면 도입할 것인가, 반복할 것인가, 포기할 것인가?

예: 가설 검증됨. 모바일 결제 프로세스 변경안을 전체 트래픽에 적용한다.

// BigQuery로 고급 분석하기
SELECT
  traffic_source.source,
  COUNT(DISTINCT user_pseudo_id) as users,
  SUM(ecommerce.purchase_revenue) as revenue,
  SUM(ecommerce.purchase_revenue) / COUNT(DISTINCT user_pseudo_id) as revenue_per_user
FROM `project.dataset.events_*`
WHERE _TABLE_SUFFIX BETWEEN '20240101' AND '20240131'
GROUP BY traffic_source.source
ORDER BY revenue DESC
